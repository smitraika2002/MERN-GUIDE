# 📘 Promise vs Asynchronous Code in JavaScript

## 📌 Introduction

In JavaScript, understanding asynchronous behavior and Promises is very important for handling tasks like API calls, timers, and file operations.

## 🔁 What is Asynchronous Code?

### 👉 Simple Definition:
Asynchronous code means JavaScript does **not wait** for a task to finish.  
It continues running other code.
